# Top 10 Budget International Destinations for Indians — 2026

A static single-page travel report ranking the 10 cheapest international trips an Indian passport holder can take in 2026, with cost breakdowns, visa info, and a practical planning timeline.

## What's inside

- 10-country snapshot grid with daily budgets and flight estimates
- 3 deep-dive itineraries (Nepal, Vietnam, Georgia) with line-item costs
- Side-by-side comparison table
- Final top-5 ranking with reasoning
- 12-week pre-trip planning timeline
- Money-saving hacks that actually work

## Tech

- Plain HTML + inline CSS — no build step, no framework, no JavaScript
- Deployed on Vercel as a static site

## Local preview

Just open `index.html` in any browser. There is no build step.

## Deploy

Pushes to `main` auto-deploy via the connected Vercel project.
